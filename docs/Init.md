จาก wireframe รูปนี้สร้างเว็บ Visualize 2D ข้อมูลการเมืองโดยมีทั้งหมด 4 Section
1. Party Clustering (อธิบายภาพรวมว่าแต่ละพรรคมี Character ยังไงด้วยการทำ Clustering ตาม Score ที่เรากำหนด)
2. Spider Chart ของแต่ละพรรค (มี Dropdown ให้เลือก)
3. Stacked Bar Chart 4 กลุ่มกลุ่มละ 4 แท่ง (แสดงแบบ 2 Row) Row1:แสดงข้อมูบของพรรคที่เราเลือกใน Dropdown ข้อที่ 2 Row2: กลุ่ม stackbar 3 กลุ่มของพรรค top 3 โดยที่ในแต่ละกลุ่ม 4 แท่งแสดงข้อมูล Performance ของแต่ละพรรคดังนี้ 1.จำนวนครั้งการลงมติ 2. จำนวนครั้งการ Multi-task (พรรคนั้นมีคนที่เป็นรัฐมนตรี ละไม่ว่างเข้ามาลงมติงี้ เขาก็ทำงานอยู่แต่แค่หลายงานเฉย ๆ ไม่ได้อู้) 3.จำนวนกฏหมายที่ร่างแล้วผ่าน 4.จำนวนกระทู้ที่เปิดอภิปราย
4. Comment Section (แสดง Comment ของแต่ละพรรคที่เราเลือกใน Dropdown ข้อที่ 2 โดยแสดงเป็น List ที่มีการจัดเรียงตามวันที่ และมีการแสดงชื่อผู้พูดด้วย)

Style ของเว็บจะเป็นแบบ minimal 2D ใช้สีที่สบายตาโดยสีของการ Visualization ทำให้เหมาะกับสีของพรรคนั้นๆ โดยที่สีของพรรคจะถูกกำหนดตามข้อมูลที่เรามีอยู่แล้ว เช่น พรรค A สีแดง พรรค B สีฟ้า เป็นต้น และใช้ Font ที่อ่านง่ายและดูสะอาดตา เช่น Sarabun เพื่อให้ข้อมูลที่แสดงออกมาดูเป็นระเบียบและเข้าใจง่ายสำหรับผู้ใช้งาน.
# Technology Stack ที่จะใช้ในการสร้างเว็บนี้คือ:
- Frontend: Next.js สำหรับการสร้าง UI และการจัดการ State ของเว็บ
- Visualization: D3.js สำหรับการสร้าง Stackbar ที่ใช้ในการแสดงข้อมูล
- Backend: Node.js สำหรับการจัดการ API และการเชื่อมต่อกับฐานข้อมูล
- Database: ในขั้นตอนนี้ยังไม่ต้องสนใจมากจะเป็นการอ่าน File CSV ที่มีข้อมูลอยู่แล้ว แต่ในอนาคตอาจจะพิจารณาใช้ฐานข้อมูลเช่น Graphql
- Style: Tailwind 4.0 โดยสร้าง global.css ใส่ Config สีที่ใช้ทุกครั้งห้าม Hardcode ค่าสีเพื่อนให้ง่ายต่อการแก้ไขในอนาคต และใช้ Font Sarabun ที่โหลดจาก Google Fonts เพื่อให้เว็บดูสะอาดตาและอ่านง่าย

# Implementation Plan:
ในส่วนของ Frontend พยายามใช้ Functional Component และ Hooks ของ React เพื่อให้โค้ดดูสะอาดและง่ายต่อการจัดการ State โดยจะมีการสร้าง Component แยกต่างหากสำหรับแต่ละ Section เช่น PartyClustering, SpiderChart, StackedBarChart, และ CommentSection เพื่อให้โครงสร้างของโค้ดดูเป็นระเบียบและง่ายต่อการบำรุงรักษา ในส่วนของ Visualization จะใช้ D3.js เพื่อสร้างกราฟต่างๆ โดยจะมีการกำหนดสีและรูปแบบของกราฟให้สอดคล้องกับ Style ที่ได้ออกแบบไว้ ในส่วนของ Backend จะใช้ Node.js เพื่อสร้าง API ที่จะรับข้อมูลจากไฟล์ CSV และส่งข้อมูลไปยัง Frontend เพื่อแสดงผลในแต่ละ Section โดยจะมีการจัดการข้อมูลอย่างเหมาะสมเพื่อให้การแสดงผลเป็นไปอย่างราบรื่นและไม่มีปัญหาเรื่องประสิทธิภาพ. API call พยายามทำให้เป็นแบบ RESTful เพื่อให้ง่ายต่อการเข้าใจและใช้งาน โดยจะมี Endpoint สำหรับแต่ละ Section เช่น /api/party-clustering, /api/spider-chart, /api/stacked-bar-chart, และ /api/comments เพื่อให้ Frontend สามารถเรียกใช้งานได้อย่างง่ายดาย ในส่วนของการจัดการข้อมูลจากไฟล์ CSV จะใช้ Library เช่น csv-parser หรือ papaparse เพื่ออ่านข้อมูลและแปลงเป็นรูปแบบที่สามารถนำไปใช้ใน Visualization ได้อย่างมีประสิทธิภาพ.

# Workflow:
โดยการทำงานของเว็บจะเริ่มจากการโหลดข้อมูลจากไฟล์ CSV ที่มีข้อมูลเกี่ยวกับพรรคการเมือง,คะแนนต่างๆ, และข้อมูลการ Cluster จากนั้นจะสร้าง Clustering และแสดงผลในรูปแบบต่างๆ ตามที่ได้ออกแบบไว้ใน wireframe โดยผู้ใช้งานสามารถเลือกพรรคที่ต้องการดูรายละเอียดเพิ่มเติมได้ผ่าน Dropdown และสามารถดู Comment ของแต่ละพรรคได้ใน Section ที่กำหนดไว้ ซึ่งทั้งหมดนี้จะถูกจัดเรียงและออกแบบให้ดูเป็นระเบียบและเข้าใจง่ายสำหรับผู้ใช้งาน.
